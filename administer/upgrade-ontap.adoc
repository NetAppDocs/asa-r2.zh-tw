= 升級 ASA R2 儲存系統上的 ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您在 ASA R2 系統上升級 ONTAP 軟體時，您可以利用新的增強 ONTAP 功能，協助您降低成本，加速關鍵工作負載，改善安全性，並擴大組織可用的資料保護範圍。

ASA r2 系統的 ONTAP 軟體升級流程與其他 ONTAP 系統的升級流程相同。建議您先link:https://docs.netapp.com/us-en/ontap/upgrade/create-upgrade-plan.html["使用 Active IQ Digital Advisor 或 Upgrade Health Checker 中的 Upgrade Advisor 為您的升級做好準備"^]。

做好升級準備後，建議使用link:https://docs.netapp.com/us-en/ontap/upgrade/task_upgrade_andu_sm.html["從 System Manager 自動進行不中斷升級（ andu ）"^]執行升級。ANDU 利用 ONTAP 的高可用性（HA）故障轉移技術，確保叢集在升級期間能夠持續不間斷地提供資料服務。

了解更多有關link:https://docs.netapp.com/us-en/ontap/upgrade/index.html["ONTAP 軟體升級"^]的資訊。



== 在計劃升級之前執行自動升級前檢查

您無需正在進行 ONTAP 軟體升級即可執行 ONTAP 自動升級預檢查。獨立於 ONTAP 自動升級程序執行預升級檢查，可讓您查看針對叢集執行的檢查項目，並取得一份錯誤或警告列表，以便在開始實際升級之前進行修正。

從 ONTAP 9.19.1 開始、錯誤和警告會分開顯示、以協助您輕鬆識別升級前必須解決的問題。

了解如何link:https://docs.netapp.com/us-en/ontap/upgrade/automated-pre-checks.html["執行 ONTAP 自動升級前檢查"^]。
