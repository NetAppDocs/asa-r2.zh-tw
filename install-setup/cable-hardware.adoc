= 為 ASA R2 儲存系統連接硬體纜線
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
為 ASA r2 儲存系統連接電纜，以啟用叢集通訊、主機網路存取、管理連線和儲存架連線。



== 硬體佈線總覽

由於每款 ASA r2 型號的插槽位置、連接埠指派和 I/O 模組配置各不相同，因此不同平台型號的佈線方式也不同。請選擇適用於您的 ASA r2 型號的程序：

* link:cable-hardware-a1k.html["Cable ASA A1K"]
* link:cable-hardware-a70-90.html["纜線 ASA A70 和 ASA A90"]
* link:cable-hardware-a20-30-50.html["纜線 ASA A20、A30 和 A50"]
* link:cable-hardware-c30.html["Cable ASA C30"]


在硬體佈線期間，您將連接四種類型的線纜：

* *叢集 / HA 連線*
+
將兩個控制器相互連接（無交換器叢集）或連接到叢集網路交換器（有交換器叢集）。這些連接可實現叢集通訊以及控制器配對之間的高可用性容錯移轉。叢集互連流量和 HA 流量共用相同的實體連接埠。

* *主機網路連線*
+
根據主機組態，使用乙太網路（iSCSI、NVMe/TCP）或光纖通道（FC、NVMe/FC）將控制器連接到 SAN 主機。這些連線可為您的儲存工作負載提供資料存取。

* *管理網路連線*
+
將控制器連接至您的管理網路，以便透過 ONTAP System Manager 和 CLI 進行系統管理存取。

* *儲存櫃連線*
+
將控制器連接至 NS224 儲存櫃，為您的系統提供儲存容量。
