= ASA R2 儲存系統的 ONTAP 軟體支援與限制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
雖然 ASA R2 系統提供廣泛的 SAN 解決方案支援、但不支援某些 ONTAP 軟體功能。

.ASA R2 系統不支援下列項目：
* iSCSI LIF 容錯移轉
* FabricPool
* LUN 密集資源配置
* MetroCluster
* 物件傳輸協定
* ONTAP S3 SnapMirror 和 S3 API
* SnapMirror （非同步與同步）移轉至雲端
* SnapMirror （非同步和同步）至非 ASA R2 系統
* 選擇性LUN對應（SLM)


.ASA R2 系統支援下列項目：
* SnapLock
* 雙層加密
