= 保護 ASA R2 儲存系統上的 Kubernetes 應用程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 Astra Control Center 來保護 Kubernetes 應用程式。Astra Control Center 可讓您將應用程式和資料從一個 Kubernetes 叢集移轉到另一個叢集、使用 NetApp SnapMirror 技術將應用程式複寫到遠端系統、並將應用程式從移轉到正式作業。
